## What Endpoint Do We Use to Generate Strategies?

The primary endpoint for generating trading strategies is:

```
POST https://api.youtrade.ai/api/v1/strategies/create
```

This endpoint accepts your trading intent in natural language and returns a complete, production-ready FreqTrade strategy.

### Endpoint Details

| Property | Value |
|----------|-------|
| **Method** | `POST` |
| **URL** | `https://api.youtrade.ai/api/v1/strategies/create` |
| **Content-Type** | `application/json` |
| **Authentication** | Bearer Token (JWT) |
| **Rate Limit** | 10 requests/day (Free), 100/day (Pro) |

### Request Body Schema

```json
{
  "name": "string (required)",
  "description": "string (required)",
  "intent": {
    "pairs": ["string"] (required),
    "timeframe": "string (optional, default: '15m')",
    "risk_tolerance": "string (optional: 'low' | 'moderate' | 'high')",
    "strategy_type": "string (optional)",
    "indicators": ["string"] (optional)
  }
}
```

<Info>
The more detailed your description, the better the AI can understand your strategy and generate accurate code.
</Info>

## How Do We Obtain an API Key and Authenticate?

### Step 1: Create an Account

Visit [https://dashboard.youtrade.ai](https://dashboard.youtrade.ai) and sign up for a free account.

### Step 2: Generate an API Key

1. Navigate to **Settings** → **API Keys** in your dashboard
2. Click **"Create New API Key"**
3. Give your key a descriptive name (e.g., "Production Bot", "Testing")
4. Select the appropriate permissions:
   - `strategies:create` - Create new strategies
   - `strategies:read` - View your strategies
   - `backtests:run` - Execute backtests
   - `deploy:create` - Deploy to cloud
5. Click **Generate**
6. **Copy and save your API key immediately** - you won't be able to see it again!

<Warning>
**Keep your API key secure!** Never commit it to version control or share it publicly. Treat it like a password.
</Warning>

### Step 3: Authenticate Requests

Include your API key in the `Authorization` header with the `Bearer` scheme:

```bash
Authorization: Bearer YOUR_API_KEY_HERE
```

### Authentication Example

```bash
curl -X POST https://api.youtrade.ai/api/v1/strategies/create \
  -H "Authorization: Bearer yt_1234567890abcdef" \
  -H "Content-Type: application/json"
```

## Example Code

### Bash (cURL)

Here's a complete example using cURL to create a strategy:

```bash
#!/bin/bash

# Set your API key (use environment variable in production)
API_KEY="yt_your_api_key_here"
BASE_URL="https://api.youtrade.ai"

# Create a new trading strategy
curl -X POST "$BASE_URL/api/v1/strategies/create" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "RSI Mean Reversion Strategy",
    "description": "Buy when RSI drops below 30 (oversold), sell when RSI rises above 70 (overbought). Use 15-minute timeframe for quick entries and exits.",
    "intent": {
      "pairs": ["BTC/USDT", "ETH/USDT"],
      "timeframe": "15m",
      "risk_tolerance": "moderate",
      "strategy_type": "mean_reversion",
      "indicators": ["rsi", "ema"]
    }
  }' \
  | jq '.'
```

### Python

Using the `requests` library for synchronous requests:

```python
import requests
import os
import json

# Configuration
API_KEY = os.getenv('YOUTRADE_API_KEY')  # Load from environment
BASE_URL = 'https://api.youtrade.ai'

# Headers
headers = {
    'Authorization': f'Bearer {API_KEY}',
    'Content-Type': 'application/json'
}

# Request payload
payload = {
    'name': 'RSI Mean Reversion Strategy',
    'description': 'Buy when RSI drops below 30 (oversold), sell when RSI rises above 70 (overbought). Use 15-minute timeframe for quick entries and exits.',
    'intent': {
        'pairs': ['BTC/USDT', 'ETH/USDT'],
        'timeframe': '15m',
        'risk_tolerance': 'moderate',
        'strategy_type': 'mean_reversion',
        'indicators': ['rsi', 'ema']
    }
}

# Make the request
response = requests.post(
    f'{BASE_URL}/api/v1/strategies/create',
    headers=headers,
    json=payload
)

# Check response
if response.status_code == 200:
    strategy = response.json()
    print(f"✅ Strategy created successfully!")
    print(f"Strategy ID: {strategy['strategyId']}")
    print(f"Status: {strategy['status']}")
    print(f"\nGenerated code preview:")
    print(strategy['code'][:500] + '...')
else:
    print(f"❌ Error: {response.status_code}")
    print(response.json())
```

### Python (Async with httpx)

For async applications:

```python
import httpx
import asyncio
import os

async def create_strategy():
    """Create a trading strategy asynchronously"""
    
    API_KEY = os.getenv('YOUTRADE_API_KEY')
    BASE_URL = 'https://api.youtrade.ai'
    
    async with httpx.AsyncClient() as client:
        response = await client.post(
            f'{BASE_URL}/api/v1/strategies/create',
            headers={
                'Authorization': f'Bearer {API_KEY}',
                'Content-Type': 'application/json'
            },
            json={
                'name': 'RSI Mean Reversion Strategy',
                'description': 'Buy when RSI drops below 30 (oversold), sell when RSI rises above 70 (overbought). Use 15-minute timeframe for quick entries and exits.',
                'intent': {
                    'pairs': ['BTC/USDT', 'ETH/USDT'],
                    'timeframe': '15m',
                    'risk_tolerance': 'moderate',
                    'strategy_type': 'mean_reversion',
                    'indicators': ['rsi', 'ema']
                }
            }
        )
        
        if response.status_code == 200:
            strategy = response.json()
            print(f"✅ Strategy created: {strategy['strategyId']}")
            return strategy
        else:
            raise Exception(f"API Error: {response.status_code} - {response.text}")

# Run the async function
if __name__ == '__main__':
    asyncio.run(create_strategy())
```

### JavaScript (Node.js)

Using the Fetch API (Node.js 18+):

```javascript
// Load API key from environment
const API_KEY = process.env.YOUTRADE_API_KEY;
const BASE_URL = 'https://api.youtrade.ai';

async function createStrategy() {
  try {
    const response = await fetch(`${BASE_URL}/api/v1/strategies/create`, {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${API_KEY}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        name: 'RSI Mean Reversion Strategy',
        description: 'Buy when RSI drops below 30 (oversold), sell when RSI rises above 70 (overbought). Use 15-minute timeframe for quick entries and exits.',
        intent: {
          pairs: ['BTC/USDT', 'ETH/USDT'],
          timeframe: '15m',
          risk_tolerance: 'moderate',
          strategy_type: 'mean_reversion',
          indicators: ['rsi', 'ema']
        }
      })
    });

    if (!response.ok) {
      throw new Error(`HTTP error! status: ${response.status}`);
    }

    const strategy = await response.json();
    
    console.log('✅ Strategy created successfully!');
    console.log(`Strategy ID: ${strategy.strategyId}`);
    console.log(`Status: ${strategy.status}`);
    console.log(`\nGenerated code preview:`);
    console.log(strategy.code.substring(0, 500) + '...');
    
    return strategy;
    
  } catch (error) {
    console.error('❌ Error creating strategy:', error);
    throw error;
  }
}

// Execute
createStrategy();
```

### JavaScript (Using Axios)

If you prefer Axios:

```javascript
const axios = require('axios');
require('dotenv').config(); // Load .env file

const API_KEY = process.env.YOUTRADE_API_KEY;
const BASE_URL = 'https://api.youtrade.ai';

const createStrategy = async () => {
  try {
    const response = await axios.post(
      `${BASE_URL}/api/v1/strategies/create`,
      {
        name: 'RSI Mean Reversion Strategy',
        description: 'Buy when RSI drops below 30 (oversold), sell when RSI rises above 70 (overbought). Use 15-minute timeframe for quick entries and exits.',
        intent: {
          pairs: ['BTC/USDT', 'ETH/USDT'],
          timeframe: '15m',
          risk_tolerance: 'moderate',
          strategy_type: 'mean_reversion',
          indicators: ['rsi', 'ema']
        }
      },
      {
        headers: {
          'Authorization': `Bearer ${API_KEY}`,
          'Content-Type': 'application/json'
        }
      }
    );

    console.log('✅ Strategy created successfully!');
    console.log('Strategy ID:', response.data.strategyId);
    console.log('Status:', response.data.status);
    
    return response.data;
    
  } catch (error) {
    if (error.response) {
      // Server responded with error
      console.error('❌ API Error:', error.response.status);
      console.error('Message:', error.response.data);
    } else {
      // Network or other error
      console.error('❌ Error:', error.message);
    }
    throw error;
  }
};

createStrategy();
```

### Response Field Descriptions

| Field | Type | Description |
|-------|------|-------------|
| `strategyId` | UUID | Unique identifier for the strategy |
| `name` | string | Name of the strategy |
| `description` | string | Your original description |
| `status` | string | Strategy status: `generating`, `ready`, `failed` |
| `createdAt` | ISO-8601 | When the strategy was created |
| `updatedAt` | ISO-8601 | Last modification time |
| `config` | object | FreqTrade configuration settings |
| `components` | array | List of components used in the strategy |
| `code` | string | Complete Python strategy code |
| `filePath` | string | Where the strategy file is saved |
| `validation` | object | Validation results |
| `metadata` | object | Generation metadata |
